# Snake Game in C++

A console-based Snake Game built using C++.

## Features

- Snake movement using WASD controls
- Collision detection
- Score tracking
- High score saving
- Difficulty levels
- Snake tail growth

## Technologies Used

- C++
- File Handling
- Windows Console API

## Controls

- W → Up
- S → Down
- A → Left
- D → Right
- X → Exit

## How to Run

1. Compile the code:
   g++ snake.cpp -o snake

2. Run the game:
   ./snake
